Classe FileSystemHelper

La classe FileSystemHelper permet de gérer les fichiers et dossiers sur le système. Elle inclut des méthodes pour vérifier l'existence d'un fichier ou d'un dossier, et pour les créer si nécessaire.


class FileSystemHelper {
    public static function ensureDirectoryExists($directoryPath, $createIfNotExist = true) {
        if (!is_dir($directoryPath)) {
            if ($createIfNotExist) {
                mkdir($directoryPath, 0777, true);
                echo "Dossier créé : " . $directoryPath;
            } else {
                echo "Dossier non trouvé : " . $directoryPath;
            }
        } else {
            echo "Le dossier existe déjà : " . $directoryPath;
        }
    }
}

Description

La méthode ensureDirectoryExists vérifie si un dossier existe à l'emplacement donné. Si le dossier n'existe pas et si l'option $createIfNotExist est définie à true, le dossier sera créé.

Fonction DirectoryChecker

La classe DirectoryChecker est utilisée pour vérifier l'existence d'un dossier et renvoyer un message selon le résultat.


class DirectoryChecker {
    public function checkDirectory($path) {
        if (is_dir($path)) {
            return "Le dossier existe.";
        } else {
            return "Le dossier n'existe pas.";
        }
    }
}

Description

La méthode checkDirectory prend un chemin de dossier en paramètre et renvoie un message indiquant si le dossier existe ou non.

Utilisation du Code

Voici comment intégrer et utiliser ces classes et méthodes dans une application PHP.


include 'FileSystemHelper.php';
include 'DirectoryChecker.php';

// Vérifier et créer un dossier si nécessaire
FileSystemHelper::ensureDirectoryExists('path/to/directory');

// Vérifier l'existence d'un dossier
$checker = new DirectoryChecker();
echo $checker->checkDirectory('path/to/directory');

Explication

Dans cet exemple, nous incluons les fichiers des classes FileSystemHelper et DirectoryChecker, puis nous les utilisons pour vérifier et créer des dossiers sur le système de fichiers.
