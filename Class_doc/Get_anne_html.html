Documentation Simple - Compte à Rebours

Ce projet est un "compte à rebours" qui aide à savoir combien de temps il reste avant un événement, comme un anniversaire ou un examen. Quand l'événement est passé, la couleur de la boîte change pour le montrer !

Comment ça marche ?

Le compte à rebours fonctionne comme une horloge qui décompte les secondes, minutes et heures jusqu'à l'événement. Si le temps est écoulé, la boîte change de couleur pour dire "C'est fini !" et l'événement est terminé.

Les parties du projet

  • PHP : Ce code calcule le temps restant avant l'événement. Exemple d'utilisation :
    • 
$date_event = "2024-12-31 23:59:59";
$compte_a_rebours = new Get_anne($date_event);
$temps_restant = $compte_a_rebours->get_temps_restant();
echo "Il reste " . $temps_restant["jours"] . " jours, " . $temps_restant["heures"] . " heures.";
  • HTML : Ce code structure la page et affiche les informations sur le temps restant. Exemple d'utilisation :
    •

  • CSS : Ce code stylise la page. Exemple d'utilisation :
    • 
#compte-a-rebours {
    font-size: 24px;
    background-color: lightblue;
    padding: 20px;
    text-align: center;
}
  • JavaScript : Ce code met à jour le compte à rebours chaque seconde et change la couleur de la boîte une fois l'événement passé. Exemple d'utilisation :
    • 
let timer = new CountdownTimer("2024-12-31 23:59:59", 1);
